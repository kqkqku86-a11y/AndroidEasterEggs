Proyek ini berisi kode lengkap untuk semua Easter egg Android, bertujuan untuk mengatur dan memastikan kompatibilitas dengan semua Easter egg Android. Tujuannya adalah untuk memungkinkan sebagian besar perangkat untuk menikmati berbagai versi Easter egg dengan modifikasi minimal pada kode Easter egg tersebut.

* **Play Store** menggunakan [Penandatanganan Aplikasi Play](https://support.google.com/googleplay/android-developer/answer/9842756). Hal ini akan menyebabkan pembaruan aplikasi dari sumber lain gagal. Anda perlu menghapus instalasi versi Play Store dan menginstal versi dari sumber lain untuk melakukan pembaruan dari sumber lain, dan sebaliknya.
* **Pgyer** menjadi host versi Alpha. Versi Alpha berisi fitur-fitur yang belum selesai dan belum final, serta dapat dihapus, dikerjakan ulang, atau diganti tanpa pemberitahuan. Versi ini mungkin mengandung bug.
